# SAP 成本分析技能

当用户提问涉及 SAP 成本分析、月结、产品成本合理性分析、CKM3、COOIS、WIP、
物料账 ML、标准成本、实际成本、差异分析、获利分析、成本中心、生产订单结算等
CO FICO 模块内容时，使用此技能。

## 触发词

SAP成本分析、成本差异、CKM3、COOIS、物料账、标准成本vs实际成本、
生产订单结算、月结分析、产品成本、WIP计算、获利分析

## 功能

支持以下分析场景：

- 产品标准成本与实际成本的差异分析，偏差阈值 >10%
- 材料成本、人工成本、制造费用、加工费的逐项差异分解
- CKM3 物料价格分析
- COOIS 生产订单查询与分析
- WIP 在制品计算与结算
- 物料账 ML 差异分摊逻辑验证
- 成本中心费用归集与分摊合理性检查
- 获利分析（COPA）维度校验
- 费用同比环比分析

## 前置条件

- 用户需要访问 SAP 系统（RFC 接口）
- IMA 知识库中需存储 SAP 相关的分析模板
- 需要安装 SAP RFC 连接器（MCP connector）
- 用户需具备 FICO/CO 模块基础知识

## 使用步骤

1. 从 IMA 知识库检索对应期间的 SAP 数据文件
2. 解析 Excel 数据，提取成本相关字段
3. 计算标准成本与实际成本的差异
4. 识别偏差 >10% 的产品
5. 按成本组件分解差异（材料/人工/制造费用/加工费）
6. 生成差异分析图表和 HTML 报告
7. 输出结论和建议

## 工具依赖

- IMA 知识库 MCP 连接器
- Python pandas（数据处理）
- matplotlib（图表生成）
- SAP RFC 接口（可选，用于实时数据拉取）

## 输出格式

- HTML 内联报表（通过 html-inline-report 技能）
- Excel 数据文件
- Markdown 分析摘要

## 注意事项

- 帕累托原则：80% 的差异通常来自 20% 的产品，优先分析 TOP N
- 成本差异分析的根因可能在物料账（ML）、生产订单（COOIS）、采购信息记录甚至 BOM 层级
- 不要只盯着最后一行差异数字，要系统性分析
- 每月 4 号开展成本分析是固定节奏
